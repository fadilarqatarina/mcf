Welcome

Theme Requirements

You must be running WordPress 5.0 or higher, PHP 7.4 or higher, and MySQL 8 or higher.


Recommended PHP Configuration Limits

max_execution_time 180

memory_limit 128M

post_max_size 32M

upload_max_filesize 32M

Theme Installation





Theme installation via WordPress

  1. Log into your WordPress Dashboard and browse to Appearance > Themes.
  2. Click Add new button, located at the top of the screen or Add new theme.
  3. Click on Upload Theme button at the top of the screen and click on Choose File, locate minfolio.zip.
  4. Click on Install Now button.
  5. Click on Activatelink.
  6. After the theme is activated, the notice will appear “This Theme requires the following plugins...”, click Begin installing plugins.

Theme installation via FTP

  1. Log into your site via FTP and browse to your folder located at wp-content/themes.
  2. Unzip the minfolio.zip file you downloaded and upload to the server in folder located at wp-content/themes.
  3. Next log into your WordPress site and navigate to Appearance > Themes.
  4. Click on Activatelink.
  5. After the theme is activated, the notice will appear “This Theme requires the following plugins...”, click Begin installing plugins.

Theme installation errors

Are you sure you want to do this?

Above error typically happens in the senarios given below.

  • You are trying to upload the Full zip file and not just the minfolio.zip.
  • Your server memory limit to PHP is too low.
  • Your upload limit isn’t large enough and you must update your PHP.INI file to increase the upload_max_filesize value or contact your webhost to fix it.

The package could not be installed. The theme is missing the style.css stylesheet.

Above happens when you try to upload the Full zip that includes documentation, demo content... etc. Please make sure you are uploading only minfolio.zip.

Demo Import

  1. Open WordPress Dashboard.
  2. Go to Appearance > Import Demo Data.
  3. Click on Import Demo Dataand wait until you see the message "That's it, all done!".

Note : Dont's close or navigation away from the browser while the import is in process.


Theme Updates

Updating the theme via WordPress is very similar to the installation process via WordPress.


Theme installation via WordPress

  1. Go to WordPress Dashboard.
  2. Go to Appearance > Themes.
  3. Deactivate Minfolio Theme by simply activating a different theme.
  4. Once you activate a different theme, you can delete the Minfolio Theme(This won’t delete your content).
  5. Now install the latest downloaded version of the theme(See Theme Installation via WordPress).

Theme installation via FTP

  1. Log into your site via FTP and browse to your folder located at wp-content/themes.
  2. Delete minfolio folder (This won’t delete your content).
  3. Download the latest version and unzip the minfolio.zip file and upload to the server in folder located at wp-content/themes (You can simply drag and drop the folder).
  4. Next log into your WordPress site and navigate to Appearance > Themes to activate Minfolio.

Note : Remember to clear your browser cache after each update.

Page & Page Builder

Create a new page

  1. Navigate to Pages in your admin sidebar and click Add New option.
  2. Enter a new name for your page and add the content. Publish the page.

Elementor Page Builder has been used to build pages in the theme. Click on the link for the documentation documentation.


Elementor settings can customized in 'Dashboard > Elementor > Settings'.

Enable 'Elementor Page Builder' to use in portfolio post by checking the 'Portfolio Items'.


Use Elementor to Build Page

  1. Choose Edit with Elementor to edit your page
  2. Choose elements you want to add to your new page in the element box and start editing them.




Homepage / Postpage Setup

  1. Go to Settings > Reading in your WordPress Dashboard.
  2. Set "Home/Index page displays" to a "Static Page".
  3. In the Homepage drop down menu choose your home page. (Note – Pages will only be available in the dropdown if they have been created).
  4. In the Blog page drop down menu choose your blog page.
  5. Click on Save changes button

Menu

Menu setup

  1. In WordPress Dashboard go to Appearance > Menus.
  2. Click on create a new menulink.
  3. Enter menu name in Menu Name text field.
  4. Click the Create Menu button.
  5. Under Menu Settings check the box Main Menu against Display Location.
  6. Click on Custom Links and add appropriate URL and Link Text
  7. Click on Add to Menu button.
  8. Once the menu links are added, it will appear on the right side under Menu Structure and can be customized, reordered.

Menu Customisation

  1. In WordPress Dashboard go to Appearance > Customize > menu.
  2. Under 'Menu settings' you can make required customisations.
  3. Click on Save button.

Portfolio Posts

Note:-Enable 'Elementor Page Builder' to use in portfolio post. 'Dashboard > Elementor > Settings > General > Post Types > Portfolio Items'

Create a portfolio post for page project.

  1. In your WordPress Dashboard, go to Portfolio Items > Add New Item.
  2. Enter Portfolio Item Title.
  3. Under Portfolio Thumbnail in Title, enter appropriate title.
  4. Under Portfolio Thumbnail in Subtitle, enter appropriate subtitle.
  5. Under Portfolio Settings in Project type choose preferred Project.
  6. In "External Project" you can add preferred external URL.
  7. You can specify where to open the link.
  8. Enter appropriate title & value for metas.
  9. Under Portfolio Metas in Social media share, check to enable social media share in portfolio page.
  10. Now click on Publishbutton

Adding portfolio section in a page

  1. Go to page where you want to add portfolio section and Edit with Elementor
  2. Drag Portfolio Sectionfrom Elementor left panel and drop on the page

Create a portfolio category

  1. In your WordPress Dashboard, go to Portfolio Items > Portfolio Categories.
  2. Enter the category name in the textfield below name (This name will be displayed in portfolio filter on the site).
  3. Click on Add New Portfolio Category button.
  4. You can also create a new category from the portfolio page itself by clicking on Add New Portfolio Category link.

Sort a portfolio post

  1. In your WordPress Dashboard, go to Portfolio Items > Sort Portfolio.
  2. Rearrange the order to sort the portfolio post simply by drag & drop.

Blog Posts

Create a blog post

  1. In your WordPress Dashboard, go to Posts > Add New.
  2. Enter Post title and Post content.
  3. Enter Excerpt if required.
  4. Choose appropriate post format on the right side under Format pane. Select image for image post, Video for video post etc. Based on your selection you will see additional fields, gallery will display drop zone to drop multiple images, video will display text field for the URL.
  5. Choose appropriate Category and Tags. New Categories can be added by clicking Add New Category link.
  6. Click Set featured image link for post image.
  7. Click Publish button.

Create blog post category

  1. In your WordPress Dashboard, go to Posts > Categories.
  2. Enter the category name in the textfield below name.
  3. Click on Add New Category.
  4. You can also create a new category from the blog page itself by clicking on Add New Category button.

Contact Form

we have created contact forms using contact form 7 plugin in this Theme. Below are the steps to create contact form.

Create new contact form

  1. In your WordPress Dashboard, go to Contact > Add New.
  2. Enter the form name.
  3. Click on Mail tab and enter the required information.
  4. Click on Save button.

Adding contact form in a page

  1. Go to page where you want to add contact form and Edit with Elementor
  2. Drag Contact Form 7from Elementor left panel and drop on the page
  3. Select the Contact form on the left panel and update the changes.


Selecting theme contact form

After the demo import is done, edit the contact page using the elementor. Select 'Agency Contact Form' from the dropdown and update the changes.
